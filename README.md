## Trader Behavior vs Market Sentiment

This project analyzes the relationship between Bitcoin market sentiment
(Fear & Greed Index) and trader performance using historical trade data.

### Key Insights
- Traders perform better during Greed sentiment
- Win rates decline during Fear periods
- Leverage usage increases during Greed
- Market sentiment influences trading behavior

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Outcome
The analysis demonstrates how sentiment-aware strategies can improve
trading performance and risk management.
